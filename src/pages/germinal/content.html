Germinal

Germinal is a separate Lean 4 conjecture-generation and proof-attempt project vendored inside this repository. ProofX keeps it isolated from the root Python engines and bounded certificate layer.

Source README

Project Boundary

Separate Toolchain

Germinal has its own Claude instructions, frontend, API, tests, and Lean workflow under packages/germinal/.

Different Role

ProofX ranks and records bounded search candidates. Germinal experiments with conjecture generation, formalization, and proof attempts.

Same Discipline

A surviving search is still only unrefuted. Neither project treats failed disproof attempts as mathematical truth.
